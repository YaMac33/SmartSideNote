🏛️行政機構図ってなに？

行政機構図は、政府や行政機関の組織構造を「見える化」したもの。
つまり、
「どんな組織があって、誰が何をしているか」がパッとわかる地図みたいなものです！

1. 行政機関全体（1.pdf）

大きな組織の「骨格」を示します。

例えば、

  • 内閣官房
  • 内閣府
  • デジタル庁
  • 復興庁

これらが連携して行政を動かしています。

2. 各府省の組織構成（2.pdf）

こちらはもう少し細かく👇

【例：総務省】
├─ 総務局 │ ├─ 地方行政部門 │ ├─ 情報通信政策部門 ├─ 消防庁 └─ 人事課

各府省に内局（内部部局）や外局があります。職員の定員も記載されていて、どのくらい人がいるかもわかります。

3. 内閣・内閣府（3.pdf）

内閣は政府のトップチーム🏆

内閣総理大臣 │ ├─ 内閣官房 │ ├─ 国家安全保障局 │ ├─ 内閣サイバーセキュリティセンター │ └─ 内閣人事局 ├─ 内閣法制局 └─ 各委員会（公正取引委員会など）

4. 内閣本府の詳細（4.pdf）

ここは政策の中枢です🔥

内閣総理大臣 ├─ 内閣官房長官 ├─ 内閣官房副長官 ├─ 内閣危機管理監 ├─ 各種本部（サイバーセキュリティ本部、健康医療戦略本部など）

👀例えば「内閣サイバーセキュリティセンター」は日本のネット安全を守る重要な部署！

5. 内閣府の外局・審議会・地方支分部局（5.pdf）

外局：独立した組織（例：金融庁、公正取引委員会）

審議会：政策を検討・議論する場

地方支分部局：地域ごとの行政サービスを担う

施設等機関：特定の業務を担当する施設

内閣府 ├─ 外局（金融庁など） ├─ 審議会（政策検討） ├─ 地方支分部局（地域行政） └─ 施設等機関（特定業務）

🌟まとめ＆ポイント

  • 行政機構図は政府の「見える地図」！
  • 組織のつながりや役割がビジュアルで理解できる
  • 政策立案や人事異動の背景が見えてくる
