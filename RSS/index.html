RSSニュースをGeminiで自動要約する方法

Google One AI Premiumを活用したGoogle Apps Script自動化

🔍 概要

このページでは、RSSフィードから最新ニュースを取得し、それをGoogleのGemini（AI）に自動で要約させてGoogleスプレッドシートに出力する方法を解説します。Google One AI Premiumに加入していれば、追加費用なくこの機能を使うことが可能です。

✅ 必要なもの

  • Google アカウント（One AI Premium加入済）
  • Google スプレッドシート
  • Apps Scriptの編集権限

🧩 ステップ1：スプレッドシート作成

Googleスプレッドシートを新規作成し、「拡張機能 → Apps Script」を開きます。

🧩 ステップ2：コードを貼り付け

以下のコードをエディタに貼り付けます：

const FEED_URL = "https://news.yahoo.co.jp/rss/topics/top-picks.xml";

async function summarizeRSSNews() {
  const feed = UrlFetchApp.fetch(FEED_URL).getContentText();
  const xml = XmlService.parse(feed);
  const root = xml.getRootElement().getChild("channel");
  const items = root.getChildren("item");

  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  sheet.clear(); // 古いデータを消す
  sheet.appendRow(["タイトル", "リンク", "要約"]);

  for (let i = 0; i < Math.min(5, items.length); i++) {
    const item = items[i];
    const title = item.getChildText("title");
    const link = item.getChildText("link");

    const prompt = `次のニュースタイトルを日本語で100文字以内で要約してください:\n\n${title}`;

    const summary = await geminiSummary(prompt);

    sheet.appendRow([title, link, summary]);
  }
}

async function geminiSummary(prompt) {
  const response = await Gemini.newTextSession()
    .withSafetySettings([
      { category: "HARM_CATEGORY_HARASSMENT", threshold: "BLOCK_NONE" },
      { category: "HARM_CATEGORY_HATE_SPEECH", threshold: "BLOCK_NONE" },
      { category: "HARM_CATEGORY_SEXUALLY_EXPLICIT", threshold: "BLOCK_NONE" },
      { category: "HARM_CATEGORY_DANGEROUS_CONTENT", threshold: "BLOCK_NONE" },
    ])
    .generateContent(prompt);

  return response.text.trim();
}

💡 補足・応用

  • RSSの本文を取得したい場合は別途HTMLパーサーの導入が必要
  • 定期実行したい場合は「トリガー」で1時間おきなどに自動化可能
  • Slack通知やLINE通知と連携も応用可能
