👶 保育士不足、どこまで深刻なのか？

1. 「保育の質どうなる？ 76年ぶり基準改善も保育士不足深刻」 🔍

保育士の配置基準、2024年度にようやく改善された（4・5歳児は30人→25人に）。…が、実際は人が足りず、単発バイトでどうにか回してる園も多いらしい。こども家庭庁も動いてるけど、制度と現場のギャップがなかなか埋まらない。

出典：日テレNEWS NNN via Yahoo!ニュース

2. 「保育士配置基準、85.4％が『少なすぎる』と回答」 📊

キッズラインの調査では、約85％の保育士が「配置基準が足りない」と回答。壁面制作や書類作成など、保育“以外”の仕事も多くて、もう手が回らない…。

出典：PR TIMES

3. 「保育政策『量』から『質』へ」 ⚖️

施設数を増やすだけじゃなくて、子どもたちの「育ち」をどう支えるか。こども家庭庁が「保育の質」を重視する方向に舵を切ってるのは期待したい。

出典：毎日新聞、福祉新聞 via Yahoo!ニュース／ほいくis

4. 「待機児童数は7年連続で減少」 📉

数字としては良い傾向だけど…「人は足りない」「休めない」「辞める人が多い」——これは本当に改善してるのか？

出典：ほいくis

👩‍🎓 未来の担い手が減っている…

5. 「保育士養成・募集、短大廃止で深刻さ増す」 ⚠️

保育士を目指す人の入り口となる短大が、定員割れなどで次々と募集停止や廃止。つまり、育てる人も減ってきてる…。これは本当にヤバいかも。

出典：Yahoo!ニュース via 福祉人事.com

🛠️ 現場を支える工夫と制度

6. 「保育ソーシャルワーカーに注目」 🤝

家庭の困りごとを支援しつつ、保育士の業務も軽くしてくれる「保育ソーシャルワーカー」。こういう専門職との連携は、今後もっと広がるといいな。

出典：Yahoo!ニュース via 福祉人事.com

7. 「保育士の賃金・待遇改善を求める声」 💬

処遇改善、雇用の安定（非正規→正規）、現場からの声は多数。でもまだ“声を上げるだけでは変わらない”という空気感もあるみたい…。

出典：Reddit（スレッド引用）

8. 「県職員の保育施設『ぎふっこ保育園』開所」 🏢

県庁内に保育園を作った岐阜県。子育てしながら働きやすい環境づくり、これも人材流出を防ぐ一歩。

出典：Yahoo!ニュース via 福祉人事.com

9. 「ICT導入で保育士の業務効率化」 📱

業務連絡や保護者とのやり取りをアプリで効率化。「CoDMON」などのツール導入で、保育士の時間が少しでも楽になるのはありがたい。

出典：福祉人事.com via newscast.jp

🔒 安全対策も進行中

10. 「送迎バス安全装置、85.7％に設置完了」 🚌

痛ましい事故を受けて、安全装置の設置が義務化。ただ、残りの14％弱が気になるところ。全台完了するまで気が抜けない。

出典：福祉人事.com via TBS NEWS DIG
