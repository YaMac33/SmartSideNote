自治体職員のための公的ポイント管理ガイド

公費で購入した物品のポイント、正しく管理できていますか？

そのポイント、私物化は重大なコンプライアンス違反です

公費による購入で得たポイントは、個人のものではなく、自治体の「財産」です。その私的利用は、法的な責任を問われる可能性があります。このセクションでは、ポイントの私的流用がなぜ深刻な問題なのかを、法的・倫理的・財務的な3つの側面から解説します。

⚖️

法的リスク

ポイントは地方自治法上の「財産（債権）」。私的流用は業務上横領罪に問われる可能性があり、管理を怠り失効させることは住民監査請求の対象になり得ます。

👤

倫理的リスク

公務員に求められる「公私の別」の原則に反します。私的利益のために職務上の地位を利用したと見なされ、懲戒処分の対象となり、住民の信頼を失います。

💰

財務的リスク

ポイントは経費削減に使える貴重な資産です。これを活用せず失効させることは、本来削減できたはずの公費を浪費していることと同じです。

正しい管理 vs 誤った管理

❌ 誤った方法：立替払による私的取得

  1. 職員が個人カードで物品を立替購入。
  2. 購入時に個人のアカウントにポイントが付与される。
  3. 職員は自治体に経費を請求し、精算を受ける。
  4. 結果：自治体の財産であるべきポイントが職員個人に移転し、公私混同、利益相反のリスクが発生。

✅ 正しい方法：法人カードによる一元管理

  1. 職員が法人カードで物品を購入。
  2. 購入時に自治体のアカウントにポイントが付与される。
  3. 支出とポイントは一元的に管理される。
  4. 結果：ポイントは確実に自治体の資産となり、透明性が確保され、経費削減に活用できる。

これは他人事ではない：実際の監査指摘事例

ポイントの不適切取得は、理論上のリスクではありません。実際に複数の自治体で監査委員から厳しく指摘されています。ここでは代表的な事例を紹介し、問題の根本原因を分析します。これらの事例は、すべての自治体にとっての警鐘です。

福井県あわら市の事例

  • 指摘内容：教育活動の物品購入で職員が立替払した際、個人のポイントカードにポイントを付与していた。
  • 監査委員の判断：公費執行の適正性を損なう「不適切な支出」の一環として明確に指摘。
  • 示唆：少額・緊急の支出に便利な「立替払」制度そのものが、ポイントの私的取得というコンプライアンス上の脆弱性を内包していることを示す典型例。

滋賀県高島市の事例

  • 指摘内容：個人のポイントカードやクレジットカード利用によるポイント取得。
  • 監査委員の判断：「経済的付加価値が個人に転化されることになり不適正」と断定。以前の監査でも指摘済みだった。
  • 示唆：口頭注意だけでは改善せず、ルールを明文化し、組織的に取り組む必要性を示す。曖昧な状態の放置は許されない。

問題の根本原因

監査事例から見えてくるのは、自治体間で共通する3つの根本原因です。

コンプライアンス遵守と利益最大化への道筋

問題の重大性を理解した上で、次に取り組むべきは具体的な解決策です。ここでは、リスクを根絶し、ポイントという資産を住民サービス向上のために最大限活用するための、実践的な枠組みと行動計画を提示します。

ステップ1：公式なルールを定める

曖昧さをなくし、全職員が従うべき統一基準を設けるため、ポイントの取り扱いに関する公式な要綱を策定します。これは監査でも推奨される最も重要な第一歩です。

【ポイント取扱要綱（要点抜粋）】

  1. 所有権の原則：公費支出で得たポイント等は、すべて自治体に帰属する。
  2. 私的利用の禁止：職員は、自治体に帰属するポイント等を私的に利用してはならない。違反は懲戒処分の対象。
  3. 報告義務：ポイント等を取得した場合は、速やかに所属長等に報告する。
  4. 一元管理：取得は原則として法人カード等で行い、残高を適切に管理する。
  5. 適正な使用：自治体の経費を直接削減する目的にのみ使用し、会計書類に記録する。

ステップ2：仕組みを「法人カード」中心へ移行する

職員による立替払を極力減らし、自治体名義の法人カードによる調達に一元化します。これにより、ポイントの所有権が明確になり、多くの問題が根本から解決します。

評価項目 立替払 法人カード
透明性低い高い
業務効率低い（精算が煩雑）高い（精算が簡素化）
内部統制弱い強い
ポイント所有権曖昧（リスク大）明確（自治体に帰属）

ステップ3：ポイントを住民サービスに還元する

管理体制を整えたら、最終目標である「公費支出の削減」に取り組みます。蓄積したポイントは、様々な形で住民サービスに還元できます。

💸

直接割引

事務用品などの購入代金から直接値引く。

📦

物品交換

防災備蓄品や公民館の備品などと交換する。

🎟️

金券交換

ギフトカード等に交換し、必要な物品を調達。

✈️

出張経費削減

マイルを将来の公務出張の航空券代に充当。

段階的実施計画

この改革は一度にすべてを行う必要はありません。現実的な3つの段階に分けて、着実に進めていくことを推奨します。

第1段階（即時実施：〜3ヶ月） ▾
  • 緊急指令の発出：全庁に対し、ポイントの私的取得・利用を禁止する通達を出す。
  • リスク評価の実施：立替払が多い部署（学校など）を「高リスク部署」として特定する。
  • 現状把握：既存の法人カード等で蓄積されているポイント残高と有効期限を確認する。
第2段階（短期的実施：〜12ヶ月） ▾
  • 公式ポリシーの策定：「ポイント等取扱要綱」を正式に策定し、施行する。
  • 研修の展開：全職員、特に高リスク部署を対象に研修会を実施する。
  • 法人カード導入の拡大：高リスク部署から優先的に法人カード導入を開始する。
第3段階（中長期的実施：継続） ▾
  • 立替払からの完全移行：原則として立替払を廃止し、法人カード等による調達に移行する。
  • 管理システムの統合：将来の財務会計システムにポイント管理機能を盛り込むことを検討する。
  • 定期的内部監査：ポリシーの遵守状況を定期的に監査し、実効性を検証・評価する。

よくある質問（Q&A）

ポイント管理に関して職員から寄せられがちな疑問や誤解について、ここで明確に回答します。正しい知識を持つことが、コンプライアンス遵守の第一歩です。

Q1. 個人カードで立て替えた場合、ポイントは個人のものでは？ ▾

A1. いいえ、そうではありません。民間企業ではカード名義人のものと解釈されることもありますが、公務員にはより厳しい「公私の別」の原則が適用されます。公費の支出という公的行為から発生した経済的利益は、その源泉が公金である以上、最終的に自治体に帰属させる必要があります。そこに「グレーゾーン」は存在しません。

Q2. 少額のポイントでも問題になりますか？ ▾

A2. はい、問題になります。問題の本質は金額の大小ではなく、公的な地位を利用して私的な利益を得るという「行為そのもの」にあります。たとえ1ポイントであっても、その私的利用は公務員倫理に反し、住民の信頼を損なう行為です。

Q3. ポイントの管理は手間がかかって面倒です。 ▾

A3. 管理は法的な義務です。ポイントは法的に「財産」と位置づけられるため、自治体にはそれを適正に管理する「受託者責任」があります。管理が面倒だという理由は、資産管理の義務を放棄する正当な理由にはなりません。むしろ、管理を怠って失効させることは資産の浪費と見なされます。

Q4. 自治体が発行するマイナポイントなどとは違うのですか？ ▾

A4. はい、全く性質が異なります。
• 受け取るポイント（このガイドの主題）：自治体が事業者から「もらう」もので、地方自治法上の「財産」。管理目的は経費削減と財政規律の維持です。
• 発行するポイント（マイナポイント等）：自治体が住民に「あげる」もので、政策目的（経済振興など）を達成するための事業です。個別の条例や国の法律に基づきます。
この二つを混同してはいけません。
