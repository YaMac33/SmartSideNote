メール送信フォームをWebサイトに設置する方法

GitHub Pagesのような静的なサイトでも、外部サービスを利用することで簡単にお問い合わせフォームを設置できます。主な方法をイラストで解説します。

方法1: Formspreeなどの外部サービスを利用する

HTMLフォームの送信先を外部サービスに設定するだけで、指定したメールアドレスに内容を転送してくれます。

  • 手順: サイトにHTMLフォームを設置し、`action`属性にFormspreeのURLを指定します。
  • メリット: 簡単に実装でき、カスタマイズ性が高い。
